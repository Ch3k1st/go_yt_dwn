# Go Video Downloader

Простой и мощный консольный инструмент для загрузки видео с YouTube и других платформ. Написан на Go, использует мощь `yt-dlp` и `FFmpeg` для обеспечения наилучшего качества.
Если возникают какие то ошибки, обновите yt-dlp.

---

## Предварительные требования

Для работы программы в вашей системе **обязательно** должны быть установлены две утилиты. Установите их одной командой в зависимости от вашей ОС:

### 1. Установка зависимостей (yt-dlp и FFmpeg)

| ОС                 | Команда для терминала                               |
| :----------------- | :-------------------------------------------------- |
| **Windows**        | `winget install yt-dlp ffmpeg`                      |
| **macOS**          | `brew install yt-dlp ffmpeg`                        |
| **Linux (Ubuntu)** | `sudo apt update && sudo apt install yt-dlp ffmpeg` |

### 2. Установка Go

Убедитесь, что у вас установлен [Go (1.18+)](https://go.dev/dl/). Проверить можно командой: `go version`.

---

## Быстрый старт (Сборка из исходников)

Если вы скачали этот репозиторий, выполните следующие шаги:

1.  **Скомпилируйте исполняемый файл:**

    ```bash
    go build -o v-down main.go
    ```

2.  **Запустите программу:**
    - **Windows:** `.\v-down.exe`
    - **Linux/macOS:** `./v-down`

---

## Инструкция по использованию

1. Запустите программу.
2. Вставьте ссылку на видео (например, `https://www.youtube.com/watch?v=...`).
3. Программа скачает видео и сохранит его в папку `downloads` с оригинальным названием.

> Если программа выдает ошибку "FFmpeg не найден", убедитесь, что путь к папке `bin` вашего FFmpeg добавлен в системную переменную **PATH**.
